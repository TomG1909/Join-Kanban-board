1. Add Task

Lets start by creating a task.
1. Add a title for the Task (required field)
2. choose the category the task is assigned to (required field)
3. Add the task description (required field)
4. choose a due date for the task
5. define the task urgency from low to high
6. choose the creator of the task
7. click the button to finally create the task

2. Backlog

In the backlog you can see a list of all created tasks by yourself or other team members
1. the task urgency
2. the person the task is assigned to (here: the creator)
3. the category the task is assigned to
4. the description of the task
5. click the trash button to delete the task

3. Board

The board gives an overview about the progress of all tasks. With drag and drop you can move the task to the next section
1. In the 'To Do' section all tasks are shown which have to be done. The created task automatically is added to 'To Do' section
2. When you or your collegues are working on the task move it to the 'In Progress' section
3.If an app/software have to be tested or is in testing drop it in the 'Testing' section
4. If you are done move it forward to the 'Done' section
5. You can delete the task by clicking on the trash button
